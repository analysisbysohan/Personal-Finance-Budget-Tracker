# Personal-Finance-Budget-Tracker
![Personal Finance Dashboard](Screenshot%202.png)


An interactive Power BI dashboard designed to monitor personal cash flow, analyze categorical expenses against target budgets, and track net savings.

### 📊 Key Features & Visualizations
* **Dynamic Monthly Filtering:** Integrated a top-level month slicer (January–December) for granular, time-period analysis.
* **Core Financial KPIs:** At-a-glance tracking for **Total Transactions**, **Total Income**, **Total Expenses**, and **Net Savings**.
* **Budget vs. Actuals Analysis:** A horizontal bar chart comparing real expenses against monthly budget limits across major categories like Loan, Rent, Utilities, and Dining Out.
* **Categorical Expense Breakdown:** A clean donut chart showing exactly where money is spent, displaying both the absolute value (in thousands) and percentage contribution.

### 💡 Data Insights Derived
* Identifies high-fixed costs (like Loans and Rent) that dominate the monthly budget.
* Flags categories exceeding or nearing budget caps (e.g., Dining Out and Entertainment) to drive better financial discipline.
